# Template-Express-Ts

This is a template for a Node.js Express server with Typescript.

## Getting Started

### Prerequisites

- Node.js
- NPM

### Installing

1. Clone the repo
2. Install NPM packages

```sh
npm install
```

3. Run the server

```sh
npm run dev
```

4. Enjoy Express with Typescript!

### Deployment

1. Build the project

```sh
npm run build
```

2. Copy the `dist` folder to your server

3. Run the server with node

```sh
node dist/index.js
```
